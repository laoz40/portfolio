## Project Overview

### What is Horus?

I built a web app to track my gym workouts. I decided to create my own because
I don't enjoy the existing solutions and my previous method of tracking
workouts (excel sheet) was extremely inefficient and didn't scale well.

I also did this for fun. This was a for-fun learning project. As a result, this
is not a vibe-coded app. I'm using AI as a teacher, and not as a tool to
generate code for me, because learning how to actually build things myself
matters to me. I want to understand how everything works. Also I'm trying not
to become useless and obsolete.
